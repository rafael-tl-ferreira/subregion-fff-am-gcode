# subregion-fff-am-gcode
Supplementary material for the article "Optimisation of variable material orientations through design subregions for additive manufacturing of fibre composites" by P.A.C. de Freitas, T.A. Dutra, and R.T.L. Ferreira. 

Includes source code .py and .gcode printer files for the experimental examples (Figs 26-29) discussed in Section 4 of the paper.

If you use this code or the methodology in your own work, please cite the article:
Freitas, P. A. C., Dutra, T. A., & Ferreira, R. T. L. (202X). Optimisation of variable material orientations through design subregions for additive manufacturing of fibre composites. [Journal Name/DOI].
