# subregion-fff-am-gcode
Supplementary material for the article "Optimisation of variable material orientations through design subregions for additive manufacturing of fibre composites" by P.A.C. de Freitas, T.A. Dutra, and R.T.L. Ferreira. Includes source code and G-code routines for the experimental examples (Figs 26-29) discussed in Section 4 of the paper.
